4 Discussion and Conclusion

QUESTION 1: Is the vaccine effective for the whole population?

H0: The vaccine is not effective. HA: The vaccine affects whether or not one contracts AIDS. In this case the p-value for the Chisq test is 0 < 0.05 so we reject the null hypothesis. We conclude that we have the evidence to support the contention that the vaccine for people with AIDS is effective.

QUESTION 2: Does the vaccine prove effective for women?

H0: The vaccine does not prove effective for women. HA: The vaccine affects whether or not a women contracts AIDS. In this case the p-value for the Chisq test is 0 < 0.05 so we reject the null hypothesis. We conclude that we have the evidence to support the contention that the vaccine for women with AIDS is effective.

QUESTION 3: Does the vaccine prove effective for men?

H0: The vaccine does not prove to be effective for men. HA: The vaccine affects whether or not men contracts AIDS. The p-value for the Chisq test is 0.0001 < 0.05 so we would reject the null hypothesis. But we conclude that the vaccine makes guys more suseptable to contracting AIDS.

QUESTION 4: Is the vaccine effective for drug users?

H0: The vaccine is not effetcive for drug users. HA: The vaccine affects whether or not drug users contracts AIDS. The p-value for the Chisq test is 0 < 0.05 so we reject the null hypothesis. We conclude that we have the evidence to support the contention that the vaccine for drug users with AIDS is effective.

QUESTION 5: Is the vaccine effective for LGBTQ people?

H0: The vaccine is inaffective in LGBTQ people. HA: The vaccine affects whether or not LGBTQ people contracts AIDS. The p-value for the Chisq test is 0 < 0.05 so we reject the null hypothesis. We conclude that we have the evidence to support the contention that the vaccine for LGBTQ people with aids is effective.
